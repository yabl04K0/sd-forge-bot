# 🎨 SD Forge Telegram Bot

Телеграм бот для Stable Diffusion WebUI Forge с полным управлением через чат.

## Функции
- 🎨 txt2img — генерация по тексту
- 🖼️ img2img — генерация по изображению
- 🎭 Выбор LoRA через кнопки
- 🤖 Смена модели
- ⚙️ Все параметры: steps, CFG, sampler, scheduler, размер
- 🌱 Повтор с тем же seed
- ⬆️ Апскейл изображений
- 🖼️ Галерея с историей генераций
- 👑 Панель администратора
- 📊 Дневной лимит запросов для пользователей

---

## Установка

### 1. Forge с API
Запусти Forge с флагом `--api`:
```
webui-user.bat  →  добавь в COMMANDLINE_ARGS: --api
```
или в командной строке:
```
python launch.py --api
```

Проверь что работает: открой http://127.0.0.1:7860/docs

---

### 2. Создай бота в Телеграм
1. Напиши @BotFather в Телеграм
2. Отправь `/newbot`
3. Дай имя и username боту
4. Скопируй **токен**

---

### 3. Узнай свой Telegram ID
Напиши @userinfobot — он покажет твой ID.

---

### 4. Настрой config.yaml
```yaml
telegram:
  token: "1234567890:ABCdef..."  # вставь свой токен
  admin_id: 123456789             # вставь свой ID

forge:
  url: "http://127.0.0.1:7860"   # адрес Forge (обычно такой)
```

---

### 5. Установи зависимости
```bash
pip install -r requirements.txt
```

---

### 6. Запусти бота
```bash
python bot.py
```

---

## Структура файлов
```
sd_forge_bot/
├── bot.py          # Основной файл бота

├── database.py     # База данных (SQLite)
├── keyboards.py    # Кнопки и меню

├── requirements.txt

```

## Команды бота
| Команда | Описание |
|---------|----------|
| /start | Главное меню |
| /settings | Параметры генерации |
| /lora | Выбор LoRA |
| /model | Выбор модели |
| /gallery | Галерея изображений |
| /help | Справка |

Просто напиши промпт — бот сгенерирует картинку!

Добавь `--seed 12345` в промпт для конкретного seed.
